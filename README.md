# Excel-Finisher

## 概要

エクセルファイルを仕上げるためのプラグインツール

## 機能

- ブック内の複数シートの倍率を統一する。
- 印刷設定を一発で変更する。
- 非表示シートを検査する。
- ふりがなの情報を削除する。
- 定義した名前のうち、参照先が無いものは削除する。

## 必要要件

- Microsoft Excel

## 使い方

1. マクロ実行。START

## インストール

1. PERSONAL.XLSB を下記に配置する。
C:\Users\<Username>\AppData\Roaming\Microsoft\Excel\XLSTART\
